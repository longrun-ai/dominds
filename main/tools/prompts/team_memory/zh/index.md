team_memory 是 Dominds 的**团队共享记忆工具集**，用于管理团队成员之间共享的记忆：

- **共享性**：记忆对所有团队成员可见
- **持久化**：记忆会持久化到磁盘，在对话重启后仍然保留
- **协作性**：支持团队成员之间的信息共享

## 与 memory 的区别

| 特性     | memory             | team_memory        |
| -------- | ------------------ | ------------------ |
| 可见性   | 仅当前智能体       | 所有团队成员       |
| 使用场景 | 个人偏好、临时笔记 | 团队约定、知识共享 |
