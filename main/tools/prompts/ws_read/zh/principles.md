### 核心概念

#### 1. 运行时工作区（rtws）

运行时工作区（Runtime Workspace，简称 rtws）是 Dominds 运行时的工作目录，用于存储对话相关的文件和数据。

#### 2. 只读限制

ws_read 工具集仅提供读取功能，不能修改或删除文件。

**优势：**

- 避免误操作导致文件损坏
- 提高安全性
- 简化权限管理

#### 3. Ripgrep 搜索

Ripgrep（rg）是高效的文本搜索工具，支持正则表达式和多种过滤选项。

**常用选项：**

- `-i`：忽略大小写
- `-n`：显示行号
- `-l`：仅显示文件名
- `-C`：显示上下文行数

### 工具概览

| 工具             | 功能               |
| ---------------- | ------------------ |
| list_dir         | 列出目录内容       |
| read_file        | 读取文件内容       |
| ripgrep_files    | 搜索包含匹配的文件 |
| ripgrep_snippets | 搜索并显示匹配片段 |
| ripgrep_count    | 统计匹配数量       |
| ripgrep_fixed    | 固定字符串搜索     |
| ripgrep_search   | 使用 rg 高级搜索   |

### 最佳实践

#### 1. 目录查看

- 使用 `list_dir` 查看目录结构
- 了解项目组织方式

#### 2. 文件读取

- 使用 `read_file` 查看文件内容
- 支持行号和行范围

#### 3. 内容搜索

- 使用 `ripgrep_snippets` 查找代码位置
- 使用 `ripgrep_count` 统计匹配数量

### 限制与注意事项

1. 只能读取 rtws 内的文件
2. 不能写入或删除文件
3. 某些系统文件可能不可访问
